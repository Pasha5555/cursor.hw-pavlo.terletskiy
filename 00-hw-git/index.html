Основні команди Git

git config --global user.name - задає ім'я користувача
git config --global user.email - задає email користувача
git init - ініціалізує репозиторій
git status - показує стан файлів у папці
git add - додає файли, які будуть закомічені
git commit - зберігає всі зміни проекту
git log - дозволяє переглянути всі коміти за їх ідентифікаторами
git checkout - переключається між гілками чи комітами
git branch - показує існуючі гілки
git push - передає закомічені файли у віддалений репозиторій
git pull - завантажує файли з віддаленого репозиторію на локальний
git merge - об'єднує гілки
git clone - копіює репозиторій

.gitignore - приховує від гіта вказані в ньому файли/папки

Advanced Git

git revert - дозволяє скасувати коміт
git reset - дає можливість видалити коміт і його історію з гілки
git rebase - об'єднує гілки, копіюючи набори комітів з однієї в другу
git cherry-pick - копіює прописані коменти в head
git stash - тимчасво зберігає всі незафіксовані зміни відслідковуваних файлів
